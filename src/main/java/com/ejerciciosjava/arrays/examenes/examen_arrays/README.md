# Examen de Arrays y Matrices

## Descripción

Este examen evalúa el manejo de arrays unidimensionales y bidimensionales en Java.

## Estructura

- **GestorArrays.java** - Clase principal con métodos a implementar
- **TestArrays.java** - Clase de prueba con ejemplos
- **ENUNCIADO.md** - Enunciado completo del examen

## Compilación y Ejecución

```bash
# Compilar
javac -d . *.java

# Ejecutar
java com.ejerciciosjava.arrays.examenes.examen_arrays.TestArrays
```

## Métodos a Implementar

1. `sumaElementos()` - Suma todos los elementos del array
2. `encontrarMaximo()` - Encuentra el valor máximo
3. `invertirArray()` - Invierte el orden del array
4. `esSimetrica()` - Verifica si la matriz es simétrica
5. `sumaDiagonalPrincipal()` - Suma la diagonal principal
6. `contarPalabrasLongitud()` - Cuenta palabras por longitud
7. `palabrasConLetra()` - Busca palabras con letra específica
8. `mediaElementos()` - Calcula la media aritmética
9. `eliminarDuplicados()` - Elimina elementos duplicados

## Dificultad

Baja-Media - Ideal para iniciarse en el manejo de arrays.
