# Examen de Control de Flujo

## Descripción

Este examen evalúa el uso de estructuras de control de flujo en Java, incluyendo condicionales, bucles y sentencias de control.

## Estructura

- **ControladorFlujo.java** - Clase principal con métodos de control
- **TestControlFlujo.java** - Clase de prueba con ejemplos
- **ENUNCIADO.md** - Enunciado completo del examen

## Compilación y Ejecución

```bash
# Compilar
javac -d . *.java

# Ejecutar
java com.ejerciciosjava.fundamentos.examenes.examen_control_flujo.TestControlFlujo
```

## Métodos a Implementar

1. `clasificarEdad()` - Uso de if-else anidados
2. `obtenerDiaSemana()` - Uso de switch-case
3. `generarSecuenciaNumeros()` - Bucles for
4. `cuentaRegresiva()` - Bucle while
5. `simularAdivinanza()` - Bucle do-while
6. `filtrarNumeros()` - Uso de break y continue
7. `tablaMultiplicar()` - Bucles con formato
8. `calcularFactorial()` - Cálculos iterativos
9. `numerosPrimos()` - Bucles anidados
10. `patronTriangular()` - Patrones con bucles
11. `validarPassword()` - Lógica compleja de validación

## Conceptos Evaluados

- Estructuras condicionales (if-else, switch)
- Bucles (for, while, do-while)
- Sentencias de control (break, continue)
- Bucles anidados
- Validación de datos
- Manejo de strings y caracteres
- Algoritmos básicos

## Dificultad

Media - Requiere buen dominio de todas las estructuras de control.
