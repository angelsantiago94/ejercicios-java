# Examen de Variables y Tipos de Datos

## Descripción

Este examen evalúa el manejo de variables, tipos de datos primitivos, operaciones aritméticas y conversiones de tipos en Java.

## Estructura

- **GestorVariables.java** - Clase principal con atributos de diferentes tipos
- **TestVariables.java** - Clase de prueba con ejemplos
- **ENUNCIADO.md** - Enunciado completo del examen

## Compilación y Ejecución

```bash
# Compilar
javac -d . *.java

# Ejecutar
java com.ejerciciosjava.fundamentos.examenes.examen_variables.TestVariables
```

## Métodos a Implementar

1. `analizarTipoDato()` - Análisis y formateo de tipos de datos
2. `realizarOperacionesAritmeticas()` - Operaciones matemáticas básicas
3. `convertirTipos()` - Conversiones entre tipos primitivos
4. `calcularAreaCirculo()` - Cálculo geométrico con constantes
5. `manipularCadenas()` - Operaciones básicas con Strings
6. `esPar()` - Verificación de paridad
7. `evaluarEdad()` - Clasificación por rangos
8. `calcularPromedio()` - Procesamiento de arrays
9. `formatearNumero()` - Formateo de salida numérica

## Conceptos Evaluados

- Declaración e inicialización de variables
- Tipos primitivos y objetos
- Operaciones aritméticas
- Conversión de tipos (casting)
- Manejo de constantes
- Procesamiento de arrays
- Formateo de salida

## Dificultad

Baja-Media - Ideal para evaluar conocimientos fundamentales de Java.
