# Examen de Strings y Manipulación de Texto

## Descripción

Este examen evalúa el procesamiento de cadenas de texto y operaciones de manipulación en Java.

## Estructura

- **ProcesadorTexto.java** - Clase principal con métodos a implementar
- **TestStrings.java** - Clase de prueba con ejemplos
- **ENUNCIADO.md** - Enunciado completo del examen

## Compilación y Ejecución

```bash
# Compilar
javac -d . *.java

# Ejecutar
java com.ejerciciosjava.strings.examenes.examen_strings.TestStrings
```

## Métodos a Implementar

1. `contarPalabras()` - Cuenta palabras en el texto
2. `contarVocales()` - Cuenta vocales totales
3. `invertirTexto()` - Invierte el texto carácter por carácter
4. `textoMayusculas()` - Convierte a mayúsculas
5. `esPalindromo()` - Verifica si es palíndromo
6. `dividirPorVocales()` - Divide texto usando vocales como separador
7. `reemplazarVocales()` - Reemplaza vocales por carácter
8. `eliminarEspaciosExtra()` - Elimina espacios múltiples
9. `contarPalabrasLongitudPar()` - Cuenta palabras de longitud par
10. `palabraMasLarga()` - Encuentra la palabra más larga

## Dificultad

Media - Requiere conocimiento de métodos de la clase String y manejo de caracteres.
